zuppDelivery.herokuapp.com

This system was my final year individual project in Universiti Kebangsaan Malaysia and I developed it as a newbie to the technologies and programming tools that I used. zuppDelivery.herokuapp.com is an instant delivery system that help people to deliver item anytime and anywhere with help of official registered rider. It consists of 3 types of user classes which are Customer, Rider and Administrator

.
Customer
  • Register into the system
  • Log into the system
  • Create delivery order
  • Edit profile information
  • Submit report
Rider
  • Register into the system
  • Log into the system
  • Accept customer's order
  • Edit profile information
  • Update delivery status
Administrator
  • Log into the system
  • Approve Rider Registration
  • Terminate existing Rider
  • View report

The system is not a complex system which would fulffil all the criteria needed for a delivery system. However, through the development process it helped me to explore more in programming field. The programming tools that I used to develop the system were;

Programming Tools
  • PHP
  • HTML
  • CSS
  • Laravel Framework
  • MySQL
  • Xampp
  • Visual Studio Code
  • Heroku Server

MesraUKM

MesraUKM mobile application is a portal for Universiti Kebangsaan Malaysia's new intake students. It provides the information of orientation week schedule for the new intake students.
